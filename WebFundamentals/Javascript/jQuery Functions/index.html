Sgt James Doakes' Magical School of jQuery

.click

.click let's you trigger an event from a click! Click the button to make MAGIC happen!

.hide

.hide does EXACTLY what it sounds like!

.show

.show will show you things. Try clicking on Show me! to reveal a nice surprise.

.toggle

.toggle nicely combines .hide and .show so you can switch between hide and show!

.slideDown

.slideDown is like .show except it'll display with a sliding motion you can control.

.slideUp

.slideUp is like .hide except it'll hide with a sliding motion you can control.

.slideToggle

.slideUp is like .show except it'll display with a sliding down motion.

.fadeIn

.fadeIn will fade things back in!

.fadeOut

.fadeOut will fade things back out!

.addClass

.addClass will add a class of your choosing to a selected element!

.before

.before will add things before the selected element's tag

.after

.after will add things after the selected element's tag

.append

.append will let you add things BEFORE the tag ends

.html

.html either GETS you the HTML content of the selected element or SETS the HTML content of said element

.attr

.attr can GET or SET the value of an attribute for set of matched elements

.val

.val can GET or SET the values of an element

.text

.text can change the text inside an element but NOT HTML elements like .html can

.data

.data can change the text inside an element but NOT HTML elements like .html can
